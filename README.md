# Event Finder

## About the Project

Event Finder is my project idea for helping people find local activities and meet new people in a new area.

For example, if someone has recently shifted to a new locality and wants to play cricket, badminton, go for a walk, join a cycling group, or attend a bhajan, they can use this application to find events happening nearby.

The main goal is to make it easier for people to connect through small local activities.

## How It Will Work

A user who wants to organize an activity can add:

* Event name
* Activity type
* Venue
* Date and time
* Basic event details

Other users can find events in two ways:

* By typing their location manually
* By using their current location

They can then view nearby events and join the ones they are interested in.
